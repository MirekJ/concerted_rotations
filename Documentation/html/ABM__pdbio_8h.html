Takes an ordered array of atom parameters and fill them from a _pdb_atom_list structure.

Takes separate arrays for each column in pdb (coordinates, atom index, atom name, residue name etc.) and fill them in a _pdb_atom_list. If data for alternative location are not supplied (*altLoc==NULL), then empty space is leaved in pdb. If data for atom indices are not supplied (*serial==NULL), then atoms are numbered from 1.

Note
parameters in all arrays have to be ordered.

Takes separate arrays for each column in the pdb format (coordinates, atom index, atom name, residue name etc.) and fill them in pdb_atom_list.
