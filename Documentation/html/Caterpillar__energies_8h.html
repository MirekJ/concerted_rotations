Function calculate Caterpillar implicit solvatation energy of one residue.

Function substract number of contacts of given residue bond with other residues from treshold. Treshold value being larger for hydrophobic residues, which if not burried (large number of contacts) would produce energy penalty scaled by hydrophopbicity of residue *Hydrophob. LT Tested OK 11/07/17 ( reproduces energy of a protein in input. Mind the prefactors!)

Note
Function iterate only on given protein chain, so that in case of multiple proteins this will not calculate solvation energy of whole system but only within one protein!
