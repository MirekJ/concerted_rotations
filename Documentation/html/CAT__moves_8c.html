Finds a point on the c.r. manifold starting from a point on its tangent space, using a root-finding algorithm.

Parameters
[in]bb_instarting configuration (following D-H convention)
[in]anglefree variable
[in]delta_movesize of the proposed displacement along the tangent space in bb_in (in radians).
[out]bb_outnew configuration (following D-H convention)
Returns
GSL error code with the status of the computation. Returns GSL_SUCCESS if a valid bb_out was found, some error code otherwise.
Note
A rescaling of the step depending on the variable type (length or angle) should be considered in general, as specified in DOI:10.1371/journal.pone.0118342. Here we do not, as we only move angles.
